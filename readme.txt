==  With MWB CF7 Integration With Mautic Plugin, You Can: ==

* Connect to your Mautic account with the help of a Client id, Client secret ID, and Mautic base URL.
* Sync your contact form data over your Mautic account.
* Select the contact form fields & map them with Mautic object fields.
* Map a particular form field with multiple object fields.
* Select the type of object (Mautic module) you want to associate with the form.
* Create conditional filters with the help of AND / OR logic, to sync data over Mautic only if it fulfills the conditional logic.
* Trash a feed or put it in sandbox mode without page reloading.
* Enable/disable logs i.e create logs of all the API interactions with Mautic.
* Delete all data of the plugin from the database on plugin uninstall
* Enable/ disable email notification if any error occurs while sending any sort of data over Mautic.
* Delete data from the logs after every n days.
* Go through the instructions to create an app over Mautic.
* Review the count of all contact forms data synced.


== WHAT'S INCLUDED IN MWB CF7 INTEGRATION WITH MAUTIC PLUGIN? == 

**1) Integrate CF7 with Mautic Account Hassle-free**
Mautic CF7 Integration plugin builds a bridge between the CF7 and Mautic accounts. It provides a smooth and hassle-free integration environment to share the data derived from CF7 form submissions with your Mautic account. 


**2) Associate Form Fields with Mautic Modules**
The CF7 Integration with Mautic plugin maps your contact form data like name and phone number to Mautic modules like contact using the association rules called feeds


**3) Filter CF7 Submissions as per Feeds**
This feature of Mautic CF7 Integration plugin allows the administrator to filter the Contact form 7 submissions according to user input. The MWB CF7 Integration with Mautic plugin uses AND/OR logic to filter the form submissions and send the data to your Mautic account


**4) Detailed Logs with Data Sync Details**
CF7 Integration plugin for Mautic provides a detailed log including each data synced between the CF7 and Mautic. There is a logging of all the API interactions with Mautic for better error handling. The data sync logs feature has additional settings for the admin to clear, delete, and/or download logs.


* **Delete Logs After n Days**
The admin has the option with the  Mautic CF7 Integration plugin to enable the auto-delete functionality for the data sync logs. This means that the logs will be cleared after the specified number of days in the settings. 

* **Clear and Download Logs**
This feature of Mautic cf7 integration plugin allows the admin to clear the data sync logs manually, as per the requirement or in case of wrong data sync. The admin also has the option to download the data logs into spreadsheets for creating backups or sharing purposes. 


**5) Error Email Notification For Admin**
This feature of the Mautic CF7 Integration sends an Email notification to the admin if any input error occurs in the process of data sending entries over to Mautic. This way, the admin gets notified of any slight error in real-time.


**6) Delete or Uninstall Plugin Data**
The CF7 Integration with Mautic plugin gives full power to the admin to monitor the data sync over your Mautic account. It even allows the admin to delete or uninstall the data synced over CRM through a plugin.

== Frequently Asked Questions ==

= What is the basic purpose of using the Mautic CF7 Integration plugin? =
The CF7 Integration with Mautic plugin sends all CF7 data submissions to Mautic CRM as per the form field mappings and classifies them on the basis of various objects and Mautic modules assigned. 

= How to Authorize the Connection of Your Mautic Accounts? =
The admin needs to have the Mautic account or can directly sign up using the link we have provided in our plugin in the Mautic Account tab and then proceed with selecting between the Production and Sandbox Environment. Then enter the Consumer Key, Consumer Secret & Callback URL after generating it following the instruction shared in a plugin. The instructions can be seen by clicking the plus icon in the top right corner.

= How to integrate Mautic CRM with Contact Form 7 plugin? =
You can Integrate Contact Form 7 with your Mautic account by simply entering Mautic API credentials to integrate Contact Form 7 with your Mautic accounts. Enabling MWB CF7 Integration With Mautic plugin creates a sub-menu in the name of Mautic on the Contact Form. The admin can sign-up or log in to his Mautic CRM account and connect the CF7 and Mautic CRM.

= Does the CF7 Mautic plugin maintain logs for all the form data? =
Yes, We have provided this functionality in our Plugin to collect the logs. For enabling logs: Go to Dashboard > Contact > Mautic > Settings. Switch the button ‘Enable logs’ to enable logging of all the form data to be sent over Mautic and save the settings.

= Can we filter Contact Form 7 entries with this CF7 Integration with Mautic? =
Yes, the CF7 Submissions filtering option is available according to the user inputs. Filtering contact forms will be made possible by Contact Form 7 entries of AND / OR logic, sent to Mautic CRM based on user input.

= Can we update the already existing entry over Mautic CRM? =
Yes, the admin can update existing entries over Mautic CRM with the help of the “Primary key”.
